Here the elements are laid out one after the other. Just as they are placed in the HTML code. Elements that appear one after the other are block elements. Like the header tag and the paragraph tags. Elements that appear next to each other, like the text in a paragraph are all inline elements.
We have already seen about block & inline elements before. The point being these elements automatically enforce and provide this layout just like that. You dont have to do anything about it.
In most cases, like text displays or forms, this normal layout is the basic layout that you'd want (perhaps some styling tweaks at best). For more complex layout, you'll have to modify and adjust this normal layout with other options we will see below.
